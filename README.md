## 🎮 操作指南

- **点击/触摸**：选择宝石
- **滑动**：交换相邻宝石
- **连接3个**：基础消除
- **连接4个**：创建闪电宝石
- **L/T型**：创建炸弹宝石
- **连接5个**：创建彩虹宝石

## 💡 小贴士

1. 从底部开始消除，可以触发更多连锁反应
2. 保留特殊宝石，在关键时刻使用
3. 注意步数限制，合理规划每一步
4. 每日签到不要忘记，奖励很丰厚！
