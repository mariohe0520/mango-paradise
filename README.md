## ✨ 特色功能

### 🎮 核心玩法
- 经典三消机制，流畅的操作体验
- **100+ 精心设计的关卡**（从60关扩充至100关）
- **10 大主题章节**：艾尔文森林、西部荒野、暴风城、诅咒之地、燃烧平原、诺森德、翡翠梦境、虚空风暴、时光之穴、永恒之战
- 丰富的特殊宝石：闪电、炸弹、彩虹
- **限时关卡** - 在倒计时内完成挑战
- **收集关卡** - 收集指定类型和数量的宝石

### 🏡 庄园Buff系统 ⭐NEW
- **种树获得永久Buff**，庄园不再是装饰，是实打实的装备栏！
  - 🌟 **金芒树** → 开局自带1个炸弹宝石
  - 🌙 **月光树** → 每关多2步移动
  - 🌈 **彩虹树** → 4消就出彩虹宝石（原需5消）
  - 💖 **幸福度>200** → 分数永久1.2倍
- 完成关卡、种树、签到都能增加庄园幸福度

### ⚡ 觉醒技能条 ⭐NEW
- 消消乐中每次消除充能技能条
- 充满后点击释放大招！
- 大招类型由庄园里派遣的精灵决定：
  - 🧚 **芒果仙子** → 随机变出3个炸弹
  - 🐝 **蜜蜂精灵** → 清除一整行
  - 🦋 **彩虹精灵** → 消除场上最多的同色宝石
- 精灵在庄园中解锁和切换

### 👿 Boss血条战斗 ⭐NEW
- **10个Boss关**（每章节末尾1个）
- Boss有血条UI，每次消除=攻击Boss（伤害=消除得分）
- Boss会反击！
  - ❄️ **冰冻** - 冻住格子，无法操作，需相邻消除解冻
  - 🔒 **锁定** - 锁住宝石，需要匹配2次才能消除
- 过关条件从"凑分"变成"打空Boss血条"
- 从树精长老到萨格拉斯，10位史诗Boss等你挑战

## 🎮 操作指南

- **点击/触摸**：选择宝石
- **滑动**：交换相邻宝石
- **连接3个**：基础消除
- **连接4个**：创建闪电宝石（有🌈树buff则创建彩虹宝石）
- **L/T型**：创建炸弹宝石
- **连接5个**：创建彩虹宝石
- **技能条满**：点击精灵图标释放大招

## 💡 小贴士

1. 先去庄园种树！Buff能极大提升战斗力
2. 派遣合适的精灵，技能条满后及时释放大招
3. Boss关优先消除大连击造成高伤害
4. 冰冻格子需要在旁边消除来解冻
5. 锁定格子需要参与两次消除才能打开
6. 从底部开始消除，可以触发更多连锁反应
7. 每日签到不要忘记，奖励很丰厚！
